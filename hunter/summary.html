⏰ 期間・日数（ひっかけ頻出）

  • ✅ 所持許可の有効期間 → 3回目の誕生日まで ❌「3年間」は誤り
  • ✅ 更新申請期間 → 満了2か月前〜1か月前
  • ✅ 譲受け後の確認 → 14日以内 ❌「30日」は誤り
  • ✅ 住所変更の届出 → 30日以内 ❌「14日」は誤り（譲受けと逆）
  • ✅ 失効後の処分期限 → 50日以内 ❌「30日」「60日」は誤り
  • ✅ 許可後に未所持で失効 → 3か月以内
  • ✅ 所持者死亡時の返納 → 10日以内
  • ✅ 銃砲検査 → 年1回 ❌「2年に1回」は誤り
  • ✅ 考査合格証明書の有効期間 → 1年 ❌「3年」は誤り
  • ✅ 技能講習修了証明書の有効期間 → 3年 ❌3年超はどんな事情でも無効
  • ✅ 眠り銃の取消（令和7年改正後） → 2年以上不使用

📦 数量の上限

  • ✅ 実包・空包（自宅保管） → 合計800個 ❌「各々800個」は誤り！「合計」
  • ✅ 無煙火薬・黒色猟用火薬 → 合計5kg以内
  • ✅ 銃用雷管 → 2,000個以内
  • ✅ 標的射撃の無許可消費 → 1日400個以下
  • ✅ 狩猟・有害鳥獣の無許可消費 → 1日100個以下
  • ✅ 帳簿の保存期間 → 最終記載日から3年 ❌「1年」は誤り

⚖️ 欠格事由の期間

  • ✅ 禁錮以上の刑（執行後） → 5年間
  • ✅ 凶悪な違法行為 → 行為日から10年間
  • ✅ DV禁止命令 → 3年間
  • ✅ 猟銃の最低年齢 → 20歳以上
  • ✅ 空気銃の最低年齢 → 18歳以上
  • ✅ 認知機能検査が必要な年齢 → 75歳以上

🌍 やむを得ない事情の特例（超頻出）

対象：海外旅行・海外出張・災害・病気

❌ 国内旅行は対象外！

  • ✅ 条件①：交付後 3年以内 の技能講習修了証明書
  • ✅ 条件②：帰国後 1か月以内 に申請

❌「2か月以内」❌「5年以内の証明書」は誤り

→ 2条件を満たすと射撃教習なしで所持許可を受けられる

🎯 射程距離の数値

  • ✅ スラッグ弾の有効射程 → 約50〜100m　最大到達距離：約700m
  • ✅ 空気銃の最大到達距離 → 約300m
  • ✅ ライフル弾の最大到達距離 → 約4,000m

❌「45度で最大」は真空中の理論値。実弾では誤り

❌ 欠格事由（許可を受けられない人）

  • • 精神障害・認知症・薬物中毒・アルコール中毒
  • • 自殺のおそれがある者・住居不定者・暴力団員
  • • 禁錮以上の刑の執行後5年未満
  • • 凶悪な違法行為から10年未満
  • • DV禁止命令から3年未満
  • • 20歳未満（猟銃）／18歳未満（空気銃）

→「程度が軽い」「医師の診断書」「家族の同意」があっても認められない！

⚠️ 失効 vs 取消し

  • 失効：有効期間満了等による自動的な効力消滅
  • 取消し：違反行為等を理由とした行政処分による消滅

両方とも失効後50日以内に譲渡・廃棄等の措置が必要

🏛️ 申請窓口

  • • 所持許可申請 → 住所地の警察署
  • • 火薬類譲受許可 → 住所地の警察署
  • • 消費許可申請 → 消費する場所の警察署
  • • 狩猟者登録 → 猟を行う都道府県知事

  • ✅ ライフル銃・空気銃の装填上限 → 計6発（弾倉5＋薬室1）
  • ✅ 猟銃の全長 → 94cm以上、銃身長 48.8cm以上
  • ✅ 空気銃の全長 → 80cm以上
  • ✅ ライフル銃の所持要件 → 継続10年以上の猟銃所持

🎯 口径の表し方（超頻出！）

ゲージ（番）＝銃の口径

番号が小さいほど口径（直径）が大きい

12番 ＞ 16番 ＞ 20番 ＞ 410番

❌「番号が大きいほど口径が大きい」←逆！

号（ショット号数）＝散弾の粒の大きさ

号数が大きいほど粒が小さい

9号（小粒・小型鳥向き） ↔ バックショット（大粒・大型獣向き）

⚙️ 撃発機構の順序（必須暗記）

引き金 → 逆鉤 → 撃鉄 → 撃針 → 発火！
  • ✅ 引き金は「逆鉤」を作動させる（撃鉄を直接動かすのではない）
  • ✅ 所持の形態：携帯・運搬・保管の3つ（「2つ」は誤り）

🔧 主要部品まとめ

  • • 安全装置（安全子）→ 引き金の動きを止める補助。振動・衝撃には無効
  • • ベンチレーテッドリブ→ 銃身上部の溝付きレール。①陽炎逃がし②照準レール
  • • チョーク→ フルチョーク＝絞り強＝遠距離。シリンダー＝無絞り＝近距離
  • • 引き金の「遊び」→ 適度に必要。なさすぎると暴発しやすい
  • • 引き金の重さ目安→ 狩猟用：2kg以上、射撃用：1.5kg以上

🔒 ガンロッカーの基準

  • ✅ スチール板厚：1.0mm以上
  • ✅ 床または壁にボルト等で固定
  • ✅ かんぬき構造・鍵違い120種類以上

❌「車のトランク」は保管設備ではない

❌「家族に鍵を渡す」は「自ら保管」にならない

❌「同じロッカーの別引き出し」も共同保管とみなされる

🚗 携帯・運搬のルール

  • ✅ 装填の有無にかかわらず常に覆い・容器に入れる
  • ✅ 所持許可証を常時携帯（警察官の提示要求に応じる義務）
  • ✅ 許可を持たない家族に持たせることは禁止

❌「装填している場合のみ覆いが必要」←誤り

🛡️ 安全4原則（絶対に覚える！）

  • ① すべての銃は常に装填されているものとして扱う
  • ② 銃口は常に安全な方向（人のいない方向）に向ける
  • ③ 引き金には発射直前まで指をかけない
  • ④ 目標と背景の安全を確認してから発射

❌「分解して銃身のみなら銃口を人に向けてよい」←絶対NG

❌「安全装置をかければ振動しても暴発しない」←補助のみ

❌「おそらく獲物だろう」で発射→絶対禁止

⚡ 不発 vs 遅発

  • 不発：全く発射されない
  • 遅発：時間を置いて発射される

✅ 両方とも：銃口を安全方向に向けたまま10秒（30秒）待機後に慎重に開放

❌「直ちに薬室を開放する」←遅発の危険！最も危険な行為

✅ その他重要ポイント

  • • 休憩・移動時は必ず薬室を開放して実包を抜く
  • • 銃身内異物（雪・泥・葉）のまま発射→銃身破裂の危険
  • • 転倒時：銃口方向と引き金から指を離すことを優先
  • • 人に銃を渡す時：自分が確認→相手も確認→渡す
  • • 跳弾：コンクリート・岩だけでなく竹・水面・地面でも発生
  • • 肩付け＝銃床を肩と密着（反動吸収・次弾装填の確実化）

💥 実包の構造と注意事項

  • • ワッズの3機能：①推進薬と散弾の分離②ガス漏れ防止③散弾を束ねる
  • • グラム表示（例:24g）→ 装填されている散弾全体の重量

❌ 薬室長より長い実包は使用禁止→閉鎖不完全→銃身破裂

⚠️ 実包の廃棄ルール

✅ 廃棄方法：銃砲火薬店または警察署に持込

❌ 燃えるゴミとして廃棄→違法！

❌ 水で濡れた実包を乾燥させて使用→禁止

❌ 変形した実包を使用→禁止

🌿 鉛弾の環境問題

  • • 水鳥：水底の鉛散弾を砂と間違えて飲み込む→砂嚢内で溶けて鉛中毒
  • • 猛禽類：残滓の鉛弾を食べる→鉛中毒

→ 非鉛弾への切り替えが推奨・義務化

  • • 実包（ライフル用） → うち50個以下
  • • 銃用雷管 → 300個以下（ライフル用50個以下）
