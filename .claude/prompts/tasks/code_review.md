# タスク: コードレビュー

## 使い方

以下のテンプレートをコピーして、コードを貼り付けて Claude に渡す。

---

## テンプレート

```
以下のコードをレビューしてください。

【言語・フレームワーク】
Python 3.10 / FastAPI

【レビュー観点】
- [ ] 型ヒントの不足
- [ ] エラーハンドリングの漏れ
- [ ] BigQuery SQL の最適化余地
- [ ] セキュリティリスク（APIキーのハードコード等）
- [ ] パフォーマンス上の問題
- [ ] テスタビリティ

【コード】
```python
# ここにコードを貼り付ける
```

【特に気になっている点】
（任意）

問題点は重要度（High / Medium / Low）で分類し、修正案をコードで示してください。
```

---

## 出力フォーマット（Claude への指示）

```
出力形式：
1. サマリー（3行以内）
2. 指摘事項（重要度付き箇条書き）
3. 修正後コード（unified diff 形式）
```
