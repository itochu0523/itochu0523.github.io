# タスク: 仕様書作成

## 使い方

機能の概要を渡すと、仕様書の骨子を生成する。

---

## テンプレート

```
以下の機能の仕様書を作成してください。

【機能名】


【背景・目的】


【対象ユーザー / システム】


【入力】


【出力】


【制約条件】
- 言語: Python 3.10
- インフラ: GCP Cloud Run
- DB: BigQuery

【参考情報】
（既存コード、API仕様、外部ドキュメント等）

---

以下の形式で仕様書を出力してください：

# 機能仕様書: {機能名}

## 概要
## 処理フロー
## API インターフェース（該当する場合）
## データ定義
## エラー仕様
## テスト観点
```

---

## 出力例（asahidrink_labelling 向け）

ML 分類システムの出力フィールド仕様を書く場合：

```
【機能名】ラベリング出力フィールド定義（v1.5.2）
【対象システム】asahidrink_labelling パッケージ
【出力】23フィールドの分類結果 JSON
```
