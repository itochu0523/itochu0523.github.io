06 — Obsidian 連携

Obsidian でどこでも編集

📱
モバイルでも閲覧
iPhone / Android の Obsidian アプリでプロンプトを参照・編集。iCloud 経由で自動同期。
🔗
双方向同期
Obsidian Git プラグインで GitHub と自動 push/pull。10分ごとに最新状態を維持。
🗂️
リンクで整理
プロンプト同士を [[リンク]] で繋げてナレッジベース化。ノートグラフで全体を可視化。
