決めるべきルール

→

Phase 1-B — 未決定事項③

maker_brand_master の複合ユニークキー — どちらを主にするか

PK 候補が 2 つある。サンプルデータで両パターンを比較する。

UK① (maker_code, brand_name) — 推奨候補

GS1 の standard_gs1_company_code を軸にする。コードベースのため表記揺れに強い。maker_name が変わっても影響なし。

✓ 機械処理向き / 安定

UK② (maker_name, brand_name) — 代替候補

名前ベースのため人間が見てわかりやすい。ただし maker_name の表記揺れが問題になる。

⚠️ 名寄せ精度に依存
